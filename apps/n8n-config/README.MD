# 🚀 n8n Self-Hosted with Supabase DB (Docker Compose)

This project sets up a self-hosted instance of [n8n](https://n8n.io) using Docker Compose, with PostgreSQL hosted externally on [Supabase](https://supabase.com).

## 📦 Features

- Uses Supabase as the external database
- Secure credential encryption with a custom key

---

## 🧾 Requirements

- [Docker](https://www.docker.com/)
- [Supabase](https://supabase.com/) account
- `.env` file with correct DB and encryption config
